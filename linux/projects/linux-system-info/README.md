# Linux System Information

## Description

A beginner Bash script that displays useful Linux system information.

## Features

- Current User
- Current Date
- Current Time
- Current Groups
- Kernel Version
- Hostname
- System Uptime
- Linux Distribution
- CPU Architecture
- Current Working Directory

## Concepts Learned

- Variables
- Functions
- User Input
- Command Substitution
- ANSI Colors

## Run

```bash
chmod +x system_info.sh
./system_info.sh
